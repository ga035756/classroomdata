專題說明 

我們小組專題開發了一個旅遊資訊網，功能涵蓋查詢餐廳、景點以及住宿資訊，以及討論區功能，能夠讓用戶交流資訊，並透過行程規劃功能，將所有上述功能查詢到的資訊及行程，編輯成屬於自己的行程。

使用技術:

- 使用 CSS 、Bootstrap 進行切版
- 使用 React 框架完成前端架構
- 使用 Laravel 框架 ，對MySQL資料庫資料進行新增、修改、存取等功能，並寫成 api 形式供前端串接
- 前端串接 RestfulAPI 接收後端資料，實現前後端分離
- 使用 Leaflet 地圖套件配合 mapbox 服務實作地圖功能
- 使用 CKEditor 文章編輯套件來處理討論區文章編輯功能
- 使用 FullCalendar 套件進行行程編輯與分享
- 使用Googole第三方登入api , 設計進行快速登入，註冊功能
